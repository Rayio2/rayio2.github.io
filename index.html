用CSS画一些有趣的图片

Mar 27, 2018

之前用CSS画对话框的时候画过三角形，于是想着用CSS画一些奇怪的图案。
在CSS-TRICKS上有许多类似的图案，我复制了一个阴阳太极图的图片，想利用CSS做出来。
首先是画一个大的圆，利用 border-radius: 50%; 可以做到这一点，利用 http://www.colorzilla.com/gradient-editor/ 可以生成上黑下白的background-color。然后就有两种方法画里面的小圆，一种是画一大一小的两个圆，另一种则使用BORDER去形成一个环形，这和之前画三角形的时候用到的方法差不多。总的来说还是比较简单的，最后加一个转动效果即可了。
代码在我的Github上有：https://rayio2.github.io/blog/yinyang/yinyang.html

HTML空元素

Mar 18, 2018

根据MDN上的说明，一个空元素（empty element）可能是 HTML，SVG，或者 MathML 里的一个不可能存在子节点（例如内嵌的元素或者元素内的文本）的element。众所周知的，HTML的规范中定义了每个元素所能够包含的内容，因此许组合是没有任何语义含义的。比如一个 <audio> 元素嵌套在一个 <hr> 元素里。由于MDN中对于<audio> 允许内容的规定：
如果元素包含 src 属性：零个或多个 <track> 元素，其后紧跟不包含 <audio> 或者 <video>媒体元素的透明内容。
或者：零个或多个 <source> 元素，其后紧跟零个或多个 <track> 元素，其后紧跟不包含 <audio> 或者 <video>媒体元素的透明内容
因此这种嵌套没有任何的意义。
HTML中有以下空元素：
<area>
<base>
<br>
<col>
<colgroup> when the span is present
<command>
<embed>
<hr>
<img>
<input>
<keygen>
<link>
<meta>
<param>
<source>
<track>
<wbr>

如何用hexo搭建博客

Mar 6, 2018

1、在搭建博客之前，我们需要在github上新建一个新的repo,命名为rayio2(用户名).github.io，配置好repo的SSH KEY
2、然后建立一个安全且可用的目录用来放置Hexo文件
3、npm install -g hexo-cli ，安装Hexo
4、hexo init myblog ，创建myblog文件夹
5、cd myblog ，change dictionary
6、npm i
7、hexo new xxx（xxx为md文件名），然后会返回一个md文件的路径
8、cd xxx（转到md文件的目录）
9、start xxx.md，编辑md文件，内容为博客内容
10、cd myblog，start _config.yml，编辑网站配置
11、npm install hexo-deployer-git –save
12、hexo deploy，推送到geihub
13、进入对应的repo，打开github pages预览博客
