用CSS画一些有趣的图片

Mar 27, 2018

之前用CSS画对话框的时候画过三角形，于是想着用CSS画一些奇怪的图案。
在CSS-TRICKS上有许多类似的图案，我复制了一个阴阳太极图的图片，想利用CSS做出来。
首先是画一个大的圆，利用 border-radius: 50%; 可以做到这一点，利用 http://www.colorzilla.com/gradient-editor/ 可以生成上黑下白的background-color。然后就有两种方法画里面的小圆，一种是画一大一小的两个圆，另一种则使用BORDER去形成一个环形，这和之前画三角形的时候用到的方法差不多。总的来说还是比较简单的，最后加一个转动效果即可了。
代码在我的Github上有：https://rayio2.github.io/blog/yinyang/yinyang.html
